# Church Prayer App - Bug Reporting

## About This Repository

This repository is for public bug reporting for the Church Prayer App. Users can:
- View current issues and known bugs
- Track progress on bug fixes and feature requests
- Report new issues they encounter
- Stay informed about app updates and improvements

## About the Church Prayer App

The Church Prayer App provides a free way for church congregations to share prayer requests and updates. This app is intended to be a church aid, bringing the congregation together every day of the week through shared prayer and fellowship.

### Key Features
- Share prayer requests with your congregation
- Receive updates on ongoing prayer needs
- Stay connected with your church community throughout the week
- Free to use for all church members
